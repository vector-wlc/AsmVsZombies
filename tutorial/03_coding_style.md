# 编码风格

**注意，本页教程看不懂是什么意思无所谓，因为需要一些 C++ 基础，但是用着用着自然而然就懂了**

为了避免与其他库的标识符冲突，本框架所有的全局名称空间标识符都含有一个 `A`(类、函数、常量) 或者 `a` (对象、变量)，例如

``` C++

// C++ 使用双斜线作为注释，例如这行文字就是一个注释
// 在这里不需要了解下面的语句是什么意思，只需要记住本框架的命名规范

// 函数为大驼峰命名法并加有前缀 A
AShovel(3, 4); 
ACard(AXRK_1, 3, 4);

// 类为大驼峰命名法并加有前缀 A
ACobManager
AIceFiller

// 变量或对象为小驼峰命名法并加有前缀 a
aCobManger
aIceFiller
```
